# Политика конфиденциальности
**Majestic RP Marketplace Bot**

**Дата вступления в силу:** 2024  
**Последнее обновление:** 2024

---

## 📋 Введение

Данный документ описывает политику конфиденциальности Discord-бота Majestic RP Marketplace Bot (далее — «Бот»). Используя Бота, вы соглашаетесь с условиями данной политики.

---

## 📊 Какие данные мы собираем

Бот собирает и хранит следующие данные:

### 1. **Идентификаторы Discord**
   - **User ID** — уникальный идентификатор пользователя Discord
   - **Guild ID** — уникальный идентификатор сервера Discord
   - **Username** — отображаемое имя пользователя (тemporarily, только для логирования)

### 2. **API ключ Majestic RP**
   - Ваш персональный API ключ для доступа к маркетплейсу Majestic RP
   - Ключ предоставляется вами добровольно через команду `/setkey`

### 3. **Технические данные**
   - Время и дата запросов (для логирования)
   - Тип запрашиваемых данных (категории, серверы)
   - IP-адрес не собирается и не хранится

---

## 🗄️ Как хранятся данные

### Локальное хранение
Все данные хранятся **локально** на сервере, где запущен Бот:

- **Файл:** `data/keys.json`
- **Формат:** JSON
- **Шифрование:** Данные хранятся в открытом виде (рекомендуется ограничить доступ к файлу)
- **Резервное копирование:** Копии файлов могут создаваться автоматически

### Структура хранения
```json
{
  "guild_123456789": "server_api_key",
  "user_987654321": "personal_api_key"
}
```

### Безопасность хранения
- Файл `keys.json` содержит чувствительные данные
- Доступ к файлу должен быть ограничен
- Рекомендуется использовать права доступа `600` (только владелец)
- Бот не передаёт файл третьим лицам

---

## 🔑 Использование API ключа

### Для чего используется ключ
API ключ Majestic RP используется **исключительно** для:
- Авторизации запросов к официальному API Majestic RP
- Получения статистики маркетплейса
- Получения данных об объявлениях

### Что Бот НЕ делает с ключом
- ❌ Не передаёт ключ третьим лицам
- ❌ Не использует ключ для других целей
- ❌ Не публикует ключ в открытом доступе
- ❌ Не хранит ключ в облачных сервисах
- ❌ Не использует ключ для спама или злоупотреблений

### Безопасность ключа
- Ключ показывается пользователю **только один раз** при установке
- Ключ хранится в зашифрованном виде (рекомендуется)
- При утечке ключа используйте `/removekey` для удаления

---

## 🚫 Что мы НЕ передаём третьим лицам

Бот **не передаёт** ваши данные:
- ❌ Третьим лицам
- ❌ Другим сервисам
- ❌ В облачные хранилища
- ❌ Для рекламных целей
- ❌ Для аналитики (кроме внутренней статистики использования)

### Исключения
Данные могут быть раскрыты только:
- ✅ По официальному запросу правоохранительных органов
- ✅ Для защиты прав и безопасности владельца бота
- ✅ В случаях, предусмотренных законодательством

## 🗑️ Удаление данных

### Удаление API ключа
Вы можете удалить свой API ключ в любой момент:

**Команда для удаления личного ключа:**
```
/removekey personal=True
```

**Команда для удаления серверного ключа (только для администраторов):**
```
/removekey
```

### Полное удаление данных
Для полного удаления всех ваших данных:
1. Удалите API ключ командой `/removekey`
2. Обратитесь к владельцу бота с запросом на удаление логов

### Автоматическое удаление
Данные автоматически удаляются:
- При удалении бота с сервера (guild data)
- При блокировке пользователя (если доступен)

## 🔒 Безопасность

### Меры защиты
- Ограничение доступа к файлу `keys.json`
- Использование ephemeral сообщений для показа ключа
- Валидация API ключей перед сохранением
- Логирование всех операций с ключами

### Ваши обязанности
- Не передавайте API ключ третьим лицам
- Используйте `/removekey` при потере доступа
- Сообщайте о утечке ключа владельцу бота
- Не злоупотребляйте API Majestic RP
